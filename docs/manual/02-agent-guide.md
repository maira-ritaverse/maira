# エージェント向け 完全マニュアル

転職エージェント企業の担当者(advisor)が日々使う機能を網羅したマニュアルです。
管理者(admin)固有の操作は「管理者ガイド」を参照してください。

## ログインとダッシュボード

### ログイン

1. `https://www.maira.pro` にアクセス
2. 「ログイン」をクリック
3. メールアドレスとパスワードを入力
4. 自動で `/agency` にリダイレクト

### ダッシュボード(/agency)

朝の業務開始時に必ず開く画面です。

- **AI 利用可能残数**: 今月の組織全体の AI 利用枠を表示
- **今月の活動 KPI**: 新規顧客 / 新規推薦 / 面接 / 内定 / MA 配信 / LINE 受信
- **活動概要 KPI**: 活動中の顧客数、30 / 60 日以上対応なしの件数
- **自分のタスク**: 自分宛の未完了タスクを期限順で表示
- **直近の対応履歴**: 組織全体の直近 N 件の対応記録
- **重複候補**: 電話 / メール / 同名で重複と疑われる顧客
- **データ品質**: 未入力項目の件数(評価対象は active な顧客のみ)
- **クイックアクション**: クライアント登録、カレンダー、レポートへのショートカット
- **Mairaからのお知らせ**: 運営からのアナウンス

ダッシュボードは「今日やること」と「忘れているもの」を一目で把握する設計です。

## サイドバー構成

サイドバーの並び順:

1. ダッシュボード
2. カレンダー
3. 公式ライン(グループ:LINE / LINE 設定)
4. 顧客・求人管理(グループ:クライアント管理 / 求人管理)
5. マーケティング
6. お知らせ
7. レポート
8. 設定(個人設定 / メンバー管理 [admin のみ])

サイドバーは個人ごとにカスタマイズ可能(並び替え・グループ化・非表示)。

## クライアント(求職者)管理

### 一覧画面(/agency/clients)

3 つのビューを切り替え可能:

- **一覧テーブル**: 列をドラッグで並び替え、クリックでソート
- **カンバン**: ステータス別に列分けして可視化
- **カード**: 大きめのカードで個別情報を確認

#### 上部のアラート

- **沈黙アラートカード**: 14 / 30 / 60 / 90 日以上対応のない顧客数
- **重複候補カード**: 重複の可能性がある顧客グループ
- **データ品質カード**: 未入力項目の集計

#### 操作

- **新規登録**: 右上の「+ 新規登録」
- **CSV インポート**: AI 列マッピングで日本語ヘッダーを自動認識
- **バルクメール**: 複数選択して一斉メール送信(配信停止フラグを尊重)
- **CSV エクスポート**: 全件 / 選択行を CSV ダウンロード

### 詳細画面(/agency/clients/[id])

#### 主要セクション

- **プロフィール**: 氏名 / 連絡先 / 希望条件 / 業務メモ
- **タグ**: 組織で共有する分類タグ
- **担当者**: 自社メンバーを割当
- **タスク**: このクライアントに紐づくタスク一覧 + 追加
- **対応履歴**: 過去のやり取りを時系列で(電話 / メール / 面談 / メッセージ / メモ)
- **応募(referrals)**: 紹介中の求人と選考状況
- **成約・入金**: 成約イベントと売上記録
- **書類**: 履歴書 / 職務経歴書 / 推薦文
- **AI 状況サマリー**: 現在の状況を AI が要約(ストリーミング)
- **編集履歴**: 暗号化フィールドは「変更あり」のみ、その他は old → new で記録
- **他社エージェント利用中バッジ**: 該当時に紫色で表示

#### よく使う操作

- **タスク追加**: タイトル / 期限 / 優先度 / 担当を指定
- **対応履歴追加**: 種別(電話 / メール / 面談 等)+ 概要 + 詳細
- **応募追加**: 求人を選択 → 状態(introduced / interviewing / offered 等)を進行
- **クライアント詳細編集**: 業務メモ・希望条件・他社エージェント利用状況等を更新
- **求人共有**: LINE で求人カードを送信
- **面談予約**: 候補日 3 件を生成して LINE で送信(求職者がタップで確定)

### クライアント連携(求職者 ↔ エージェント)

求職者は自分のアカウントを Maira に持って、エージェントとの連携を承認できます。

- **未連携**: クライアントは登録されているが本人は Maira に未登録
- **招待済**: エージェントが招待メールを送信、本人の承認待ち
- **連携済**: 本人が承認、双方向の情報共有が開始
- **解除申請中**: 本人が解除申請、猶予期間内
- **解除済**: 解除が確定(エージェント承認 or 猶予期間経過の自動 cron)

連携中はエージェントが履歴書・職務経歴書・希望条件を閲覧できます。

## 求人管理

### 一覧(/agency/jobs)

- 公開中 / 募集停止 / 終了 のステータス管理
- CSV インポート / エクスポート
- 各求人ごとに紹介中の人数を表示

### 新規作成(/agency/jobs/new)

2 つの方法:

#### 方法 1: フォーム手入力

- 会社名・職種・勤務地・年収・必須スキル・歓迎条件など 18 項目
- 厚労省指針に沿った労働条件項目を網羅

#### 方法 2: PDF / 画像から AI 取り込み

- 求人票 PDF or 画像をアップロード
- Claude Sonnet 4.6(Vision)が 18 項目を自動抽出
- confidence(high / medium / low)が表示される
- 抽出結果をフォームに自動展開、不足を手入力で補完

### 求人詳細(/agency/jobs/[id])

- ヒーロー画像 / LINE 共有画像のアップロード
- AI による推薦先候補リスト
- 紹介中のクライアント一覧

## 公式 LINE 連携

### LINE トーク画面(/agency/line)

LINE 公式 OA Manager 風の 3 カラム構成:

- **左**: 会話一覧(すべて / 要対応 / 対応済 で絞り込み、検索可)
- **中央**: 選択した友達とのトーク履歴
- **右**: 連絡先詳細サイドバー

#### 中央のトーク機能

- テキスト送信(reply / push 自動判定)
- スタンプ / 画像 / 動画 / 音声 / ファイル / 位置情報の受信表示
- 求人カードの共有(画像 + 会社名 + ポジション + リンク)
- **面談候補日を 3 件提案** → 求職者タップで Meet/Zoom URL 自動発行
- 既読 / 未読の自動管理(求職者の 1:1 トークでは LINE 既読も送信)

#### 右サイドバーの操作

- **表示名の編集**: 鉛筆アイコンから上書き(LINE プロフィール名と独立)
- **プロフィール写真**
- **紐付けクライアント**: クライアント詳細へジャンプ
- **担当者割当**: 組織メンバーから選択(アバター付き)
- **タグ**: 友達単位のタグ管理
- **ノート**: エージェント内の引き継ぎメモ(求職者には見えない)
- **利用履歴**: 面談実施・確定など過去の記録
- **モバイル時**: ヘッダーに「詳細」ボタンが出てクライアント詳細へジャンプ

### 一斉配信(/agency/line/broadcasts)

新規一斉配信を作成して、テキストまたは求人カードを友達全員(or 連携済のみ)に配信。

- **配信種別**: テキスト / 求人カード
- **配信対象**: 全友達 / 連携済 / 未連携 / タグ別
- **送信タイミング**: 即時 / 予約配信
- **送信先プレビュー**: 課金通数を事前確認

### LINE 設定(/agency/line/settings)

- LINE Channel の Channel ID / Channel Secret / Channel Access Token を保存
- Webhook URL を LINE Developers に登録
- Rich Menu の設定
- 友達追加時の自動メッセージ

## マーケティング(LINE MA β)

`/agency/marketing` で LINE 自動配信シナリオを管理。

### 利用条件への同意(初回のみ)

admin が一度「有効化」を押す必要があります。利用条件 v1.0 への同意が記録されます。

### 7 つのシナリオプリセット

1. **LINE 友達追加後ウェルカム**: 友達追加から N 日後にウェルカム
2. **LINE 休眠求職者掘り起こし**: 最後の連絡から N 日後に再アプローチ
3. **LINE 登録者への面談促進**: 友達追加から N 日経過 + 面談未設定の場合
4. **LINE 面談前リマインド**: 面談 N 日前
5. **LINE 求人紹介**: 新規求人を該当者に
6. **LINE 面接後フォロー**: 面接実施 N 日後
7. **LINE 誕生日メッセージ**: 誕生日当日

各シナリオは ON/OFF + テンプレート編集 + テスト送信が可能。

### KPI(月次 / 任意期間)

- **LINE 配信数**: 5,000 通 / 月(プラン依存)に対する進捗
- **配信数 / クリック / 返信 / 応募**: 集計値
- **シナリオ別の送信実績**: 成功 / 失敗 / スキップ

クリックは短縮 URL(/r/[id])で計測、応募は配信後 7 日以内に referrals が作成された顧客数を attribution。

### 送信ログ(/agency/marketing/logs)

過去の配信を一覧 + フィルタ。CSV エクスポート可能。

## カレンダー(/agency/calendar)

月ビューで全メンバーの予定を一望。

- **面談**(緑)
- **受付**(紫)
- **タスク期限**(黄)
- **対応履歴**(青)
- **Web 面談 / Google 予定**
- **+ Google に予定を追加** ボタン

### 個人カレンダー購読(iCal)

個人設定 → 連携・アドオン → カレンダー購読 URL から取得。
Google カレンダー / Outlook / iCloud カレンダー等で購読可能。

## 面談 / 会議連携(Zoom / Google Meet)

### 連携設定(個人設定 → 連携・アドオン)

- **Zoom**: OAuth で連携、meeting:write スコープ取得
- **Google**: OAuth で連携、calendar.events スコープ取得

### 面談予約の流れ

1. クライアント詳細または LINE 会話画面から「面談候補日を提案」
2. 3 つの候補日時 + 所要時間(15/30/45/60/90/120 分)+ プロバイダ(Zoom/Meet)を指定
3. LINE で求職者に Flex Message として送信
4. 求職者が日時をタップ
5. 自動で Zoom/Meet URL が発行され、双方のカレンダーに同期
6. 招待者名にクライアント名が入る

### リマインダー

- 24 時間前 + 1 時間前に自動メール / LINE 通知
- prefs で個別 ON/OFF 可能

## AI 文書生成

### 履歴書 / 職務経歴書(クライアント詳細 → 書類)

- AI による自動ドラフト(志望動機 / 自己 PR / 要約等)
- Claude Sonnet 4.6 で生成
- 編集履歴 + バージョン管理
- PDF ダウンロード(厚労省推奨様式)

### 推薦文(/agency/recommendation-letters)

- 求人 + クライアントを指定して AI 下書き
- 組織共通のテンプレート(冒頭挨拶 / 末尾定型句)を適用
- 編集後 PDF として求人企業へ提出

### 会議録音 → 履歴書(/agency/clients/[id])

- 面談録音をアップロード
- Whisper で文字起こし
- Claude で構造化抽出 → 履歴書 / 職経歴書 / ヒアリングシートの草案を生成

## 応募管理(referrals)

- クライアント詳細 → 応募セクションで管理
- 状態遷移: introduced → interviewing → offered → placed (成約)
- 各状態変更で対応履歴が自動記録
- 7 日以上進捗が止まると Daily ダイジェストに上がる

## レポート(/agency/reports)

- **ステータス分布**: 求職者 × 紹介の現在スナップショット
- **成約・売上(月別)**: 純売上 = 成約 + 追加報酬 − 返金
- **入金 / 返金**
- **アドバイザー別パフォーマンス**(admin のみ)
- 期間: 今月 / 先月 / 任意期間

CSV エクスポート対応(クライアント / 求人 / 推薦 / 面接 / タスク / LINE 配信ログ等)。

## お知らせ(/agency/announcements)

組織内のメンバーに向けたアナウンス機能。

## 通知

### アプリ内通知(右上ベル)

- LINE 新着メッセージ
- 求職者の「興味あり」「応募を依頼」
- 応募ステータス変化
- タスク割当
- 等

### メール通知

個人設定 → 通知設定で個別に ON/OFF。

- メール通知全体マスター(OFF にすると全通知メール停止)
- 種類別 ON/OFF(7 種類)
- Daily ダイジェスト(毎朝 8 時 JST、admin のみ)

トランザクション系メール(招待 / パスワードリセット / 面談招待 / お問い合わせ受信)は本設定の対象外。

## 個人設定(/agency/settings)

### 通知設定(左カラム)

メール通知マスター + 種類別 ON/OFF。

### その他(右カラム)

- **プロフィール**: 表示名 + アイコン画像(JPEG/PNG/WebP、2 MiB 以下)
- **パスワード変更**
- **連携・アドオン**: Zoom / Google Meet / 会議録音アドオン / カレンダー購読 URL
- **推薦文テンプレート**
- **AI 利用状況**(admin のみ)
- **課金プラン**(admin のみ)

### アバター画像

- 設定すると右上ユーザーメニュー + メンバー一覧 + タスク担当 + 対応履歴記録者 + LINE 担当者表示で顔写真が出る
- 管理者がメンバーを一目で識別できる

## キャリア棚卸し(求職者向け、求職者側 /app/career)

求職者がチャット形式で AI と会話 → 強み・価値観・希望条件を構造化。
連携済の場合、エージェントが結果を閲覧可能(本人同意の上)。

## 音声面接シミュレーター(求職者向け)

求職者が 1 人で面接練習 → AI フィードバック。
連携済の場合、エージェントが履歴を閲覧可能。
