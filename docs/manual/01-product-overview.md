# Maira プロダクト概要

## Mairaとは

Maira(マイラ)は、転職エージェント企業向けの業務効率化SaaSです。クライアント(求職者)管理、公式LINE連携、面談スケジュール、AI による書類作成、進捗管理までを1つの画面で完結させます。

- 公式サイト: https://www.maira.pro
- 想定ユーザー: 中小〜中堅の転職エージェント企業(社員 5〜50 名)
- 提供形態: クラウド SaaS(Webブラウザのみ、PWA 対応)
- 価格: 月額サブスクリプション(プラン詳細は別途お問い合わせ)
- 開発: maira-ritaverse

## 3つの差別化要素

### 1. プロアクティブな伴走

エージェントが顧客対応で「忘れる・取りこぼす」を AI が能動的に防ぎます。

- 毎朝 8 時に「今日やること」を 1 通のメールで配信(Daily ダイジェスト)
- 30 / 60 / 90 日沈黙している顧客を自動ハイライト
- 7 日以上進捗が止まっている応募を検出して通知
- 面談 24 時間前・1 時間前の自動リマインダー

### 2. 音声面接シミュレーター

求職者が面接練習を 1 人で行える機能。

- iOS Safari でも動作する音声 I/O(SpeechRecognition + speechSynthesis)
- セッション履歴を暗号化保存し、過去の練習内容を参照可能
- AI による回答フィードバック

### 3. 会議録音 → AI 履歴書・職務経歴書の自動生成

面談録音をアップロードするだけで構造化された書類が完成。

- Whisper API で高精度な文字起こし
- Claude Sonnet 4.6 で構造化抽出(志望動機・職歴・スキル等)
- Puppeteer で厚労省推奨様式の PDF を生成
- Zoom Cloud Recording / Google Meet 録画 / 手動アップロードに対応

## 技術スタックの概要(顧客向けには簡潔に)

- フロントエンド: Next.js + TypeScript(モダンな高速 SPA)
- バックエンド / DB: Supabase(PostgreSQL + 認証 + Storage)
- AI: Anthropic Claude(Sonnet 4.6)+ Whisper
- 決済: Stripe
- メール: Resend
- ホスティング: Vercel

## セキュリティ

- 機密データを AES-256-GCM でサーバーサイド暗号化(会話履歴、キャリア情報、応募詳細、タスク、通知ペイロード等)
- Supabase RLS で組織を物理的に完全分離(他社データへのアクセスは不可能)
- LINE / Stripe / Zoom 等の webhook を HMAC-SHA256 で署名検証
- プライバシーポリシーで運営者の閲覧範囲を明示

## 想定するお客様

- 社員 5〜50 名規模の人材紹介エージェント企業
- 月間 50〜500 名の求職者を管理
- 公式 LINE を主要な顧客接点として活用したい
- Excel / スプレッドシートでの管理が限界に来ている

## 導入後に期待できる効果

- 連絡漏れ・対応忘れの削減(Daily ダイジェスト + 沈黙アラート)
- 書類作成時間の短縮(履歴書 1 件あたり 30 分 → 5 分以下)
- 公式 LINE での 1 人あたり対応可能人数の増加(50 名 → 200 名以上)
- 進捗状況の可視化(誰が・何を・いつまでにを 1 画面で把握)
