# Maira マニュアル(NotebookLM 用ソース)

このディレクトリには NotebookLM 等の AI ノートツールに取り込む前提で書かれた、構造化された Markdown ドキュメントが置いてあります。

## ファイル構成

| #   | ファイル                          | 内容                                                   | 対象         |
| --- | --------------------------------- | ------------------------------------------------------ | ------------ |
| 1   | `01-product-overview.md`          | プロダクト全体概要、差別化要素、主要機能、セキュリティ | 経営層・営業 |
| 2   | `02-agent-guide.md`               | エージェント(advisor)が使う全機能の操作マニュアル      | 一般メンバー |
| 3   | `03-admin-guide.md`               | 管理者(admin)の組織運営・連携・課金・メンバー管理      | 管理者       |
| 4   | `04-seeker-guide.md`              | 求職者(seeker)向けの機能ガイド                         | 求職者       |
| 5   | `05-faq-glossary-troubleshoot.md` | FAQ + 用語集 + トラブルシュート                        | 全員         |

## NotebookLM への取り込み手順

1. https://notebooklm.google.com/ にログイン
2. 「新しいノートブック」を作成
3. 「ソースを追加」→「ファイルをアップロード」
4. このディレクトリの 5 ファイルすべてを選択してアップロード
5. ソースタイトルを分かりやすく(例:「Maira 製品概要」「エージェント操作マニュアル」)
6. ノートブックのチャットで質問可能に
   - 例: 「LINE で求職者に面談候補日を送る手順は?」
   - 例: 「Daily ダイジェストが届かない場合の対処は?」
   - 例: 「管理者と一般メンバーの権限の違いは?」

## 活用例

### 営業活動

- 経営者プレゼン用に `01-product-overview.md` を中心にロードし、「3 分でプロダクト紹介をまとめて」と依頼
- お客様の業界(中小エージェント / 大手 / 特化型 等)に合わせた説明を生成

### 社内サポート / 一次受け

- 全 5 ファイルをロードし、お客様からの問い合わせをチャットに貼って AI に下書きを書かせる
- 「FAQ で類似質問はあるか」を即座にチェック

### 新規メンバー教育

- `02-agent-guide.md` をベースに、新入社員向けにステップバイステップの研修コンテンツを生成

## 更新方針

- 機能追加 / 重要な仕様変更があったら該当 .md を更新
- 大きな変更があれば NotebookLM 側のソースも再アップロード
- 用語集と FAQ は使いながら継続的に追記
