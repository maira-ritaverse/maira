# Myaira エグゼクティブサマリー(経営層向け)

## 1ページで分かる Myaira

**Myaira(マイラ)** は、転職エージェント企業向けの業務効率化 SaaS です。
クライアント(求職者)管理・公式 LINE 連携・面談スケジュール・AI による書類作成・進捗管理までを1つの画面で完結させ、エージェント1人あたりの対応可能数を従来の2倍に引き上げます。

- **公式 LP**: https://maira.pro(Xserver 上 の WordPress)
- **アプリ**: https://app.maira.pro(Next.js。 /login /signup /roi /contact /privacy 等 は こちら)
- **想定顧客**: 中小〜中堅エージェント企業(社員5〜50名)
- **提供形態**: クラウド SaaS(Webブラウザのみ、PWA 対応)
- **運営**: 株式会社Revorise

## 3つの差別化要素

### 1. プロアクティブな伴走

エージェントが顧客対応で「忘れる・取りこぼす」を AI が能動的に防ぎます。

- 毎朝8時に「今日やること」を1通のメールで配信(Daily ダイジェスト)
- 30/60/90日沈黙している顧客を自動ハイライト
- 7日以上進捗が止まっている応募を検出して通知
- 面談24時間前・1時間前の自動リマインダー

### 2. 会議録音 → AI 履歴書 / 職務経歴書 / 推薦文の自動生成

面談録音をアップロードするだけで構造化された書類が完成。

- Whisper API で高精度な文字起こし
- Claude Sonnet 4.6 で構造化抽出(志望動機・職歴・スキル等)
- Puppeteer で厚労省推奨様式の PDF を生成
- Zoom Cloud Recording / Google Meet 録画 / 手動アップロードに対応

### 3. 公式 LINE 連携 + 配信 / MA

- 1つの管理画面で個別トーク・タグ・メモ・編集履歴を集中管理
- 一斉配信(テキスト / 求人カード)、シナリオプリセット7件、クリック・応募計測
- 友達追加時の自動メッセージ・Rich Menu 等を設定

## 競合との違い(要約)

| 項目               | Myaira                                       | 一般的な汎用 CRM(HubSpot / Salesforce) | 国内エージェント特化 SaaS  |
| ------------------ | -------------------------------------------- | -------------------------------------- | -------------------------- |
| 業界特化           | 転職エージェント業務に特化                   | 汎用、カスタマイズが必要               | 業界特化だが古い UI が多い |
| AI 書類生成        | 会議録音 → 履歴書 / 職経 / 推薦文 を自動生成 | なし(別ツール組み合わせ)               | 一部対応 / 精度ばらつき    |
| 公式 LINE 連携     | 標準搭載、配信 / MA も統合                   | なし(別連携)                           | 別オプション               |
| Daily ダイジェスト | 標準搭載                                     | なし                                   | なし                       |
| 暗号化             | サーバーサイド AES-256-GCM + RLS で完全分離  | プラットフォーム標準                   | 個別に確認が必要           |
| 価格帯(月額)       | 中堅向けの手の届く範囲                       | 高い(エンタープライズ向け)             | 中位                       |
| 導入期間           | 30分〜1日                                    | 数週間〜数ヶ月                         | 数日〜数週間               |

## なぜ Myaira を選ぶのか — 3つの理由

1. **エージェント業界に特化** — 汎用 CRM を魔改造する必要がない、業務フローがそのままフィット
2. **AI が前提の設計** — 後付けではなく、会議録音から書類生成・推薦文・マッチングまで AI ネイティブ
3. **データ保護とコンプライアンス** — AES-256-GCM の暗号化、RLS による組織完全分離、プライバシーポリシーで運営アクセス範囲を明示
