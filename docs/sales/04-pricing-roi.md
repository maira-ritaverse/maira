## ROI 試算の前提

すべての試算で以下を共通前提とする(調整可能):

| 項目                        | 値       | 備考                              |
| --------------------------- | -------- | --------------------------------- |
| アドバイザー時給            | ¥3,000   | 月給50万 / 月160時間              |
| 求職者1名あたり書類作成数   | 0.5件    | 履歴書 / 職経                     |
| Myaira 導入後の書類作成時間 | 5分 / 件 | (従来30分)                        |
| 連絡漏れ削減率              | 80%      | Daily ダイジェスト + 沈黙アラート |
| 成約率向上                  | +5%      | リマインダー効果                  |
| 1人あたり対応可能数         | ×2倍     | (50名 → 100名)                    |

これは ROI 試算ページと完全に整合(https://app.maira.pro/roi)。

## 試算例 1: 小規模エージェント企業

### 想定企業

- アドバイザー: 3名
- 月間対応求職者: 60名
- 月間成約: 4件
- 平均紹介料: 80万円
- 月間連絡漏れ: 1件

### 効果(年間)

| 効果項目         | 計算                                             | 年間効果        |
| ---------------- | ------------------------------------------------ | --------------- |
| 書類作成時間削減 | (60名 × 0.5件 × 25分削減) × 12ヶ月 / 60 × ¥3,000 | ¥450,000        |
| 連絡漏れ防止     | 1件 × 80% × ¥800,000 × 12ヶ月                    | ¥7,680,000      |
| 成約率 +5%       | 4件 × 5% × ¥800,000 × 12ヶ月                     | ¥1,920,000      |
| **合計**         |                                                  | **¥10,050,000** |

### 投資対効果

- Myaira 月額(中堅プラン想定): 仮に月額10万円 = 年間120万円
- 投資対効果: **約 8倍**

## 業種別 ROI の特性

### IT エンジニア専門エージェント

- 公式 LINE のやり取りが頻繁(求職者の8割が LINE 中心)
- → **LINE 統合 + 配信** の効果が大きい
- 想定 ROI 倍率: 平均より +20%

### 医療 / 看護師専門エージェント

- 求職者の希望条件が複雑(勤務形態 / 通勤 / 子育て両立)
- → **AI マッチング + 副担当機能** の効果が大きい
- 想定 ROI 倍率: 平均より +15%

### 製造 / 工場長 / 技術者専門

- 書類フォーマットの厳格な遵守が必要
- → **AI 書類生成 + 推薦文テンプレート** の効果が大きい
- 想定 ROI 倍率: 平均より +25%

### 営業 / 経営幹部 / ハイクラス

- 推薦文のクオリティが成約に直結
- → **AI 推薦文 + 録音 → ヒアリングシート自動化** の効果が大きい
- 想定 ROI 倍率: 平均より +30%

## 導入後1年の典型的なジャーニー

### Month 1: セットアップ + 慣熟

- 既存データ移行(CSV インポート)
- アドバイザーへの操作レクチャー
- Daily ダイジェスト運用開始
- 期待効果: 体感的な業務時間削減を実感

### Month 2-3: 業務フロー再構築

- 副担当機能で組織内のチーム対応が始まる
- LINE 一斉配信 + シナリオが動き始める
- 推薦文 AI ドラフトを使いこなす
- 期待効果: 書類作成時間が見える形で削減

### Month 4-6: 数字で効果が見える

- レポートで成約数 / 売上が上向き始める
- 沈黙アラートが効いて取りこぼしが減る
- アドバイザー1人あたりの担当数が拡大
- 期待効果: ROI 倍率が試算通りに見え始める

### Month 7-12: 経営判断のデータドリブン化

- アドバイザー別パフォーマンスで個別指導
- 業種別 / 担当別の KPI でマーケ最適化
- 採用(増員)判断を Myaira のデータで実施
- 期待効果: 売上拡大、組織拡張

## 月額コストを上回る効果が出るタイミング

すべての試算企業で、**導入初月から月額コストを上回る効果**が発生します。

理由:

- 連絡漏れ防止だけで月額の数倍〜数十倍の効果
- 書類作成時間削減でアドバイザー稼働率が即向上
- 30日無料トライアル中に効果を確認した上で本契約に進める

## 月額コストの妥当性

「月額10万円〜40万円」は、エージェント1人の **月給の数%** に過ぎません。
一方、Myaira の効果は **アドバイザー0.6人分の稼働 = 月給30万円相当** を生み出します。

「AI を導入する」のではなく、「AI で人件費の何倍ものリターンを得る」投資です。
